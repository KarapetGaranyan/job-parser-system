# Инструкция по запуску и использованию

1. Установите зависимости:
   pip install -r requirements.txt

2. Запустите веб-интерфейс:
   python app.py

3. Откройте браузер и перейдите на http://127.0.0.1:5000/

4. Используйте форму поиска:
   - Введите ключевое слово, город, компанию
   - Выберите сайт для поиска: HH.ru, SuperJob или оба
   - Нажмите "Поиск"
   - После нажатия автоматически запускается обновление базы (парсинг свежих вакансий)
   - Дождитесь окончания загрузки (индикатор)
   - Просматривайте и экспортируйте результаты

5. Планировщик обновления теперь запускается автоматически при поиске с сайта. Отдельно запускать его не требуется.

6. Для CLI-режима выполните:
   python cli_search.py
   Далее следуйте подсказкам.

---

# Примечания
- Все вакансии сохраняются в базу данных vacancies.db.
- Для работы с SuperJob необходим актуальный API-ключ (уже прописан в коде).
- Для остановки приложения используйте Ctrl+C.
- Для запуска через Docker используйте docker-compose.yml

---

Если появятся ошибки — скопируйте их и отправьте разработчику для помощи!
