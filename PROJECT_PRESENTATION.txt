# Проект: Система парсинга вакансий с HH.ru и SuperJob.ru

## Описание
Система автоматически собирает вакансии с HH.ru и SuperJob.ru, сохраняет их в базу данных и предоставляет удобный веб-интерфейс для поиска, фильтрации и экспорта.
**Теперь обновление базы и парсинг свежих вакансий запускается автоматически при каждом поиске на сайте.**

## Основные возможности
- Поиск вакансий по ключевым словам, городу, компании
- Выбор источника: HH.ru, SuperJob или оба сразу
- Автоматическое обновление базы при поиске (без ручного запуска планировщика)
- Экспорт результатов в CSV
- Современный веб-интерфейс (Bootstrap, AJAX, индикатор загрузки)
- CLI-режим для поиска через терминал
- Фильтрация и сортировка по зарплате

## Архитектура
- Python, Flask (backend + frontend)
- SQLAlchemy + SQLite (база данных)
- Парсеры через API (hh_api.py, superjob_api.py)
- Веб-интерфейс (templates/index.html, Bootstrap)
- CLI-скрипт (cli_search.py)
- Асинхронный запуск парсинга при поиске (threading)

## Как пользоваться
1. Установить зависимости: pip install -r requirements.txt
2. Запустить веб-интерфейс: python app.py
3. Открыть http://127.0.0.1:5000/ и воспользоваться поиском (выбрать сайт, ввести параметры, нажать "Поиск")
4. После нажатия на "Поиск" база автоматически обновится и покажет самые свежие вакансии
5. Для CLI: python cli_search.py и следовать инструкциям

## Для кого
- Соискатели, желающие быстро находить вакансии с разных сайтов в одном месте
- HR-специалисты для мониторинга рынка труда

## Преимущества
- Удобство, скорость, отсутствие дублей
- Гибкая архитектура для расширения
- Современный внешний вид и UX
- Не требует ручного запуска парсера — всё делается автоматически при поиске!

## Использованные технологии
- **Python** — основной язык программирования
- **Flask** — веб-фреймворк для backend и frontend
- **requests** — для HTTP-запросов и работы с API
- **BeautifulSoup** — для парсинга HTML-страниц (HH.ru)
- **SQLAlchemy** — для работы с базой данных (ORM)
- **SQLite** — встроенная база данных для хранения вакансий
- **SuperJob API** — официальный API для получения вакансий
- **HTML, CSS, JS** — для пользовательского интерфейса
- **threading** — для асинхронного поиска на backend
